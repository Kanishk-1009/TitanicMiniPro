## 📌 Project Overview
This project uses **Machine Learning** to predict whether a passenger survived the Titanic disaster.  
It demonstrates a complete ML pipeline including **data preprocessing, feature engineering, model training, and evaluation**.

## 📊 Dataset Information
The dataset contains passenger details such as:
- Age  
- Gender  
- Ticket Class  
- Fare  
- Embarked Location  

📁 Files used:
- `train.csv`
- `test.csv`
- `gender_submission.csv`
